# Wander & Wonder

## Official Motion Blueprint

### Version 1.0

---

# Purpose

This document defines the artistic direction of motion throughout Wander & Wonder.

Its objective is not to describe implementation.

Its objective is to define how movement should feel.

Every animation, transition and microinteraction must reinforce the emotional narrative established by the project.

This guide complements:

* DESIGN_SPEC.md
* WIREFRAME.md
* IMPLEMENTATION_02_ENGINEERING.md

It never replaces them.

---

# Motion Philosophy

Motion is storytelling.

Nothing moves because it looks impressive.

Everything moves because it communicates.

Every animation should answer one question.

> **What emotion should the user feel next?**

If movement does not reinforce the story, it should not exist.

---

# Core Principles

## Invisible Craft

Animations should never call attention to themselves.

Users should remember the experience.

Not the animation.

---

## Emotional Rhythm

The website constantly alternates between:

Contemplation

↓

Discovery

↓

Energy

↓

Calm

↓

Reflection

↓

Inspiration

Motion must follow exactly the same rhythm.

---

## Natural Movement

Objects should behave as if they possess weight.

Acceleration should feel organic.

Nothing should teleport.

Nothing should snap unnaturally.

---

## Cinematic Timing

Movement should feel edited.

Not programmed.

Every transition should resemble the pacing of a documentary rather than a presentation.

---

## Layered Depth

Motion should reinforce depth.

Foreground.

Midground.

Background.

Nothing should move at identical speed unless intentionally synchronized.

---

## Purposeful Restraint

When in doubt:

Animate less.

Not more.

Premium experiences rarely depend on excessive movement.

---

# Global Motion Language

Throughout the experience movement should remain:

Calm.

Elegant.

Confident.

Deliberate.

Organic.

Refined.

Never chaotic.

Never playful.

Never exaggerated.

Never distracting.

---

# Motion Hierarchy

User attention should always follow this order.

Primary visual.

↓

Headline.

↓

Supporting content.

↓

Interaction.

↓

Decorative elements.

Decorative motion must never compete with important information.

---

# Scroll Experience

Scrolling should feel like turning the pages of a beautifully printed travel journal.

Not navigating a website.

The user should never feel rushed.

Large visual moments should naturally encourage small pauses.

# Transition Principles

Transitions should feel inevitable.

Never surprising.

Every section prepares the next one before it appears.

Large visual changes should happen gradually.

Abrupt cuts should remain exceptional.

---

# Environmental Motion

The page should always feel alive.

Never static.

Possible ambient elements include:

Cloud movement.

Light variation.

Atmospheric particles.

Birds.

Airplanes.

Fog.

Water reflections.

Vegetation.

Dust.

Lens light.

Every ambient element must remain secondary.

Users should notice them subconsciously.

Not consciously.

---

# Microinteractions

Every interaction should communicate responsiveness.

Hover effects should feel tactile.

Buttons should feel slightly lighter.

Images should suggest depth.

Links should feel alive.

Inputs should communicate clarity.

Nothing should bounce.

Nothing should wobble.

Nothing should seek attention.

---

# Motion Density

The experience should breathe.

Every highly animated sequence must be followed by calmer visual moments.

Silence is as important as movement.

---

# Motion Consistency

Similar actions should always produce similar responses.

Different components may have different personalities.

They should never belong to different products.

The entire experience must speak a single visual language.

---

# Performance Awareness

Movement should never compromise smoothness.

It is preferable to remove an animation entirely than deliver one that stutters.

Fluidity always has priority over quantity.

---

# Reduced Motion

When reduced motion is requested:

Preserve storytelling.

Remove spectacle.

Maintain visual continuity.

Replace movement with timing, composition and opacity whenever possible.

The emotional journey must remain intact.

---

# Absolute Rules

Never animate for decoration alone.

Never animate every element.

Never compete with photography.

Never distract from reading.

Never break visual rhythm.

Never repeat the exact same entrance pattern continuously.

Never prioritize technical complexity over emotional clarity.

---

# Final Rule

Before approving any animation ask one question.

> **If this animation disappeared, would the story lose emotional meaning?**

If the answer is yes, the animation belongs.

If the answer is no, remove it.
