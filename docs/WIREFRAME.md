# WIREFRAME.md

# Wander & Wonder

> Wireframe Narrativo Oficial
>
> Este documento define exclusivamente la estructura de la Home.
>
> No especifica estilos, colores, tipografía ni animaciones técnicas.
>
> Su objetivo es congelar el orden, la jerarquía y la función de cada escena antes de comenzar el desarrollo.

---

La navegación debe sentirse continua.

Nunca como una sucesión de bloques independientes.

Cada escena debe enlazarse naturalmente con la siguiente.

El usuario debe sentir que está recorriendo una historia.

---

# Principio Fundamental

Durante aproximadamente el **80% del recorrido**, la web inspira.

Solo durante el **20% final** presenta los servicios de Wander & Wonder.

Primero se despierta el deseo de viajar.

Después se ofrece la forma de hacerlo realidad.
